Embed your twitter account on your site and grow your visitors.

== Description ==

Twitter tweets and feeds makes it easy for the users to show their twitter feeds into their site. This is a simple though powerful plugin. Customizable embedded Tweet colors from the WordPress admin interface. Customizable widget. You can choose a widget theme color, enable/ disable thumbnail, enable/disable touch, customize height of the widget, add follow button and many more.

**Features:**

* Clean UI
* Themes
* Customizable widget settings
* Shortcodes
* Single Post on any Post
* Widget Images Popup
* Caching Enabled
* Touch Scrollable for Widgets
* Well Documented

Documentation on http://deepenbajracharya.com.np/documentation/twitter/1.0/

Tweet using TwitterAPI github repository: https://github.com/techies23/twitter-using-twitterapi

== Installation ==

1. Add the Twitter plugin to your WordPress installation
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

### How to Connect my Twitter Account

Goto Settings page in settings -> Twitter Settings and configure your keys here.

### How to Generate Keys

Goto https://apps.twitter.com/ to generate the keys. Create a New app and generate keys from here. https://apps.twitter.com/app/new

Follow here http://deepenbajracharya.com.np/documentation/twitter/1.0/#faq
